提示

如果你在使用过程中遇到了 bug，可以 提一个 issue。

# 主题特点

主题很大程度上保持了 Vuepress 默认主题的风格，并在此基础上添加了大量功能与优化:

# 页面增强

  • 为 Markdown 添加了更多语法，丰富文档与博客写作

    • Tex 支持
    • 自定义对齐
    • 流程图
    • 标记
    • 上下角标

  • 图片预览，支持放大，拖拽，滑动浏览，分享和下载

  • 文章信息展示

    • 阅读量统计
    • 作者与写作日期
    • 自动生成的字数与预计阅读时间
    • 标签与分类

  • 评论系统

  • 路径导航

  • 页面图标

  • 自定义页脚

# 功能增强

# 零配置自动实现并允许你定制

  • 全新主题色，允许你在浏览过程中动态切换

  • 深色模式，允许你手动切换或者是根据设备设置自动应用

  • 全屏按钮

  • PWA 支持

  • 返回顶部按钮

  • 一键复制代码

  • 复制时添加版权信息

# 更多强大功能

  • SEO 增强

  • Sitemap 生成

  • 加密特定文章或路径

  • Typescript 支持

# 样式优化

  • 图标支持

  • 默认主页优化

    • 特性拥有全新动画并支持跳转
    • 多个动作按钮
    • 宽屏布局优化

  • 其他内置组件优化

    • 导航栏
    • 侧边栏
    • 徽章 <MyBadge />: 在官方基础上增加了颜色支持

# 博客部分

  • 全新博客主页布局

  • 社交关注按钮与页面社交分享

  • 文章展示

    • 支持置顶的文章列表
    • 分类分组列表
    • 标签分组列表
    • 时间线

# 内建插件

本主题还包含了以下内建插件，如果有需要，你也可以单独进行使用或搭配其他主题。

  • @mr-hope/vuepress-plugin-comment: 评论与文章信息功能

  • @mr-hope/vuepress-plugin-component: 提供一些开箱即用的插件

  • @mr-hope/vuepress-plugin-copy-code: 提供一键复制代码块功能。

  • @mr-hope/vuepress-plugin-last-update: 支持多语言格式的最后更新时间

  • @mr-hope/vuepress-plugin-pwa: PWA 支持

  • @mr-hope/vuepress-plugin-reading-time: 阅读时间与字数统计

  • @mr-hope/vuepress-plugin-seo: SEO 增强插件

  • @mr-hope/vuepress-plugin-sitemap: 为你的站点生成 sitemap

  • vuepress-plugin-add-this: 让网站支持社交分享与关注

  • vuepress-plugin-md-enhance: 提供更多 Markdown 语法

  • vuepress-plugin-photo-swipe: 使网站的图片支持缩放与滑动浏览
