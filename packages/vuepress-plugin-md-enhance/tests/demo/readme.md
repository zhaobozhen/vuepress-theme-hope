## 使用插件

### 安装

```bash
npm i -D vuepress-plugin-md-enhance
```

### 使用

```js
// .vuepress/config.js
module.exports = {
  plugins: [
    [
      "md-enhance",
      {
        // your options
      },
    ],
  ],
};
```
